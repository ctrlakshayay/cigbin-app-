# CigBin – AI Powered Cigarette Waste Detection System

## Overview

CigBin is an AI-powered smart waste management system designed to reduce cigarette butt littering. The system uses a trained YOLO object detection model to detect cigarette butts and encourages responsible disposal through a smart reward-based mechanism.

When a cigarette butt is detected and dropped into the smart bin, the system verifies the action and rewards the user through a connected mobile/web application.

## Problem Statement

Cigarette butts are one of the most common forms of litter worldwide. They contain toxic chemicals and take years to decompose, causing severe environmental damage.

Traditional waste bins cannot track or incentivize proper cigarette disposal.

## Solution

CigBin combines **Artificial Intelligence + Smart Waste Management** to detect cigarette waste and promote responsible behavior.

The system:

* Detects cigarette butts using a trained YOLO model
* Verifies disposal through a smart bin camera
* Connects with an application to reward users for proper disposal

## Features
